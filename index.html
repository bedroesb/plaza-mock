What is PLAZA?

PLAZA is an access point for plant comparative genomics, centralizing genomic data produced by different genome sequencing initiatives.

Available PLAZA instances

Latest

2021

Version 5.0 of the PLAZA platform. This instance is focused on dicot plants.

Latest

Monocots PLAZA 5.0

2021

Version 5.0 of the PLAZA platform. This instance is focused on monocot plants.

Latest

Diatoms PLAZA 1.0

2020

This PLAZA version is an instance with a focus on diatoms.

Latest

pico-PLAZA 3.0

2020

This PLAZA version is the new default instance to focus on eukaryotic micro-organisms. A wide selection of species was selected, from multiple branches of the tree of life; e.g. green algae, brown algae and diatoms. Some landplants were also added as reference.
